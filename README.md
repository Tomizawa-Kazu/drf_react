# React + Django REST Framework Todoアプリ

### Frontend
- React
  - npmで開発(仮想環境はつかっていない)

    下記コマンドで起動する
    ```
    npm start
    ```
    起動後、下記URLで確認できる
    ```
    http://localhost:8000
    ```

### Backend
- Django REST Framework
  - Dockerでの開発

    下記コマンドで起動する
    ```
    docker-compose up
    ```
    起動後、下記URLで確認できる
    ```
    http://localhost:8005
    ```

### その他
  - react-basicとreduxは学習用に使用したため、アプリに関係なし
